## copy setting file

> cp /mnt/c/Users/<Username>/AppData/Roaming/Code/User/settings.json .
> cp /mnt/c/Users/<Username>/AppData/Roaming/Code/User/keybindings.json .
> cp /mnt/c/Users/<Username>/.vscode/extensions/extensions.json .
